# 🎧 AI Call Assistant (Agentic Voice AI)

An agentic voice AI system that can listen, reason, and respond like a real customer service phone call.

This project demonstrates how LLMs, tools, and memory can be combined to create a real-time, voice-based AI agent.


## Features

- Live microphone input
- Speech-to-Text (real-time)
- LLM-based reasoning with memory
- Tool usage (e.g., fetching data, confirming actions)
- Natural Text-to-Speech responses
- Multi-turn, context-aware conversations


## What It Demonstrates

- Agentic AI (LLM + tools + memory)
- Real-time voice interaction system
- End-to-end AI pipeline (input → reasoning → output)
- Applied conversational AI design


## Tech Stack

- Python
- OpenAI APIs:
  - Whisper (Speech-to-Text)
  - GPT-4o-mini (Reasoning)
  - TTS (Text-to-Speech)
- LangChain (Agents + Tools)
- LangGraph (State & Memory)


## Example Use Case

### 🏨 Hotel Food Ordering System

- User speaks: "I want to order a veg burger"
- AI processes intent
- Suggests add-ons or confirms order
- Handles modifications like:
  - "Add extra cheese"
  - "Cancel fries"
